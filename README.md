# ethereum-gasless-transactions
Project on Ethereum Gasless Transactions for CZ4153 Blockchain Technology

## Run locally
```console
$ cd gsn-dapp

// Terminal 1 (etherum contract)
$ npx ganache-cli

// Terminal 2 (relayer)
$ npx oz-gsn run-relayer

// Terminal 3 (front-end)
$ cd client
$ npm start
```
